# LA SLAM Website

Static multi-page site for the LA SLAM nonprofit. Includes Home, About, Programs, Events, and Contact pages with shared styling and lightweight animations.

## Pages

- Home: index.html
- About: about.html
- Programs: programs.html
- Events: events.html
- Contact: contact.html

## Project Structure

- css/styles.css: global styles and theme
- js/main.js: scroll reveal animation

## Run Locally

Open any HTML file directly in a browser, or use a local server if you prefer:

```bash
python -m http.server
```

Then open http://localhost:8000 in your browser.
